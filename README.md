### What I learned

Don't forget to add hover and media functionality.

```css
h1:hover {
    color: #F4D04E;
}
```

```css
@media screen and (max-width: 375px) {
    h5, h6 {
        font-size: 12px;
    }

    h1 {
        font-size: 20px;
    }

    p {
        font-size: 14px;
    }

    main {
        width: 277px;
    }
    
}
```
